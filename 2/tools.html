AI Tools You Can Try

ChatGPT

ChatGPT is an AI chatbot that can answer questions, explain ideas, and help you write or study. It uses machine learning to understand your words and respond in a human-like way.

Try ChatGPT

Google Gemini

Gemini (previously Bard) is Google’s AI tool that gives creative and factual answers using real-time web data. It can also summarize information from multiple websites.

Try Gemini

Microsoft Copilot

Copilot helps users with tasks in Word, Excel, and PowerPoint by creating text, analyzing data, or summarizing notes all using AI built into Microsoft apps.

Try Copilot

Perplexity AI

Perplexity is an AI search engine that gives you short and accurate answers along with sources. It’s great for doing quick research safely and clearly.

Try Perplexity

Canva Magic Studio

Canva Magic Studio uses AI to help design posters, presentations, and videos quickly. It can turn ideas into graphics and text automatically.

Try Canva Magic
